# 🎮 Video Game Sales Analysis (End-to-End Data Project)

## 🚀 Problem Statement

Analyze global video game sales data to identify top-performing genres, platforms, and regions to support business decisions.

---

## 🛠️ Tech Stack

* Python (Pandas)
* MySQL
* Power BI

---

## 📂 Dataset

* ~16,000 records
* Fields: Game, Platform, Year, Genre, Sales

---

## ⚙️ Approach

1. Cleaned and processed raw data using Python
2. Designed relational schema in MySQL
3. Executed SQL queries (JOIN, GROUP BY, aggregations)
4. Built Power BI dashboard for KPI tracking
5. Developed Streamlit app for interactive analysis

---

## 📊 Key Insights

* Action genre dominates global sales
* Who contributes highest revenue
* Peak sales observed between time

---

## 💻 powerbi

* Interactive filtering by genre, region, and year
* Dynamic visualization of sales trends
